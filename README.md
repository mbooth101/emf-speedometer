# Speedometer

Strap your Tildagon to your Hacky Racer, radio-controlled Henry Hoover, karaoke tuk-tuk, or anything else that needs a speedometer.

Written as a demo for my event-based GPS Hexpansion firmware concept, and as such requires the presence of a GPS Hexpansion such as that made by [The Machine Shop](https://themachineshop.uk/product/gps-hexpansion/).
